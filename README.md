# PSF Editora, Consultoria e Serviços Tecnológicos

Site estático pronto para GitHub Pages.

## Publicação
1. Envie todos os arquivos para a raiz do repositório.
2. Mantenha o arquivo `CNAME` na raiz.
3. Em GitHub Pages, use `main` e `/ (root)`.
4. Após o DNS validar, marque `Enforce HTTPS`.

## Estrutura
- `index.html` — página inicial
- `autor.html` — página do autor
- `assets/css/style.css` — estilos e modo claro/escuro
- `assets/js/script.js` — menu, tema e animações
- `assets/js/matrix.js` — fundo animado em canvas
- `assets/favicon/` — favicon e ícones
- `assets/img/` — logos PSF
